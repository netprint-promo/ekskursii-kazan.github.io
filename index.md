# Экскурсии по Казани

Казань — один из самых красивых городов России. Здесь мечети стоят рядом с православными храмами, а татарская кухня соседствует с европейскими кафе.

Лучший способ познакомиться с городом — **экскурсия с местным гидом**. Не автобусный тур, а живой рассказ человека, который знает каждый переулок.
